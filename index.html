ScuffedBrawlstars

by: Andrew Zhang

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I liked the game brawlstars a lot and I felt like it would be a cool challenge to make.

2. What I Learned

Everything about the game was awesome except the bots were kinda... really bad.

3. Architecture of My Game

Different classes for each "Brawler" and a bot class that used their code.

4. Technical Implementation

One thing that I am most proud of is the healing and ammo. These 2 parts took an incredibly long time to implement and there were a lot of bugs but in the end, it worked. We started by making timers for both and made limits then added the artwork to it after we got the code working in numbers.

5. Most Challenging Part

The coding of the bots. Although in the end the bots were still really bad, they work sometimes. The hardest part about the bots was making them actually attack the player as sometimes they would just stop.

6. Future Fixes and Updates

Fixing the bots. I would like to fix how they don't recognize walls and will just sit there and try to shoot through them.
